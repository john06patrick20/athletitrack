About AthletiTrack

Project Vision

AthletiTrack is a comprehensive, web-based platform designed to modernize athlete management at Eastern Visayas State University (EVSU). Our mission is to empower coaches and athletes with the data-driven tools they need to track performance, streamline event scheduling, and achieve their full potential.

Core Features

  • Role-Based Access Control: Secure, separate interfaces for Administrators, Coaches, and Athletes.
  • Team-Centric Management: A robust system for organizing athletes and coaches into specific teams based on sport, campus, and gender.
  • Automated Event Reminders: An intelligent notification system using email to ensure participants are always informed about upcoming events.
  • Game-by-Game Statistics: A dynamic "Game Report" system for coaches to enter sport-specific performance data for each athlete after an event.
  • Comprehensive Analytics: An advanced reporting dashboard with interactive charts, high-level KPIs, and data export capabilities.
